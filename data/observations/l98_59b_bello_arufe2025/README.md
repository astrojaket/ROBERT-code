# L 98-59 b transmission spectra

The two spectra in this directory are the published combined JWST/NIRSpec
G395H reductions accompanying Bello-Arufe et al. (2025), *Evidence for a
volcanic atmosphere on the sub-Earth L 98-59 b*.

reduction, matching the POSEIDON analysis associated with Figure 8. The
Firefly reduction is retained for independent pipeline comparisons.
