# Transmission convergence validation

This directory is generated by:

```bash
python examples/benchmark_transmission_convergence.py
```

The benchmark compares ROBERT target-bin correlated-k transmission against
native opacity sampling from the local ExoMolOP H2O POKAZATEL R=15,000 file.
It separates molecular-opacity convergence from the full H2O+CIA+Rayleigh
calculation and independently sweeps pressure layers and impact quadrature.

The source file is verified as SHA-256
`ba9d455db93809661aee79e7d322fde19da0236af1c1a9bb68f5b24735efb7fb`
and identifies DOI `10.1093/mnras/sty1877` and line list
`1H2-16O__POKAZATEL`.

`transmission_convergence.json` is the machine-readable record,
`transmission_convergence.csv` is the flat sweep table, and the PNG files show
the convergence and wavelength-resolved residuals.
