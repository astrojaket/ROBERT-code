# Historical HAT-P-32b Benchmarks

These files contain ROBERT's initial HAT-P-32b forward-model and retrieval
checks. They predate the versioned YAML task workflow and are retained only as
a historical record.

The archive includes the original scripts, configuration modules, notebook,
data bundle, documentation, Slurm launcher, bundle-building utility, retired
test, and locally generated outputs. Paths and external integrations are not
maintained, and this material is excluded from the active test and benchmark
suites.

Use the maintained PICASO and petitRADTRANS benchmarks in `examples/` when
validating current ROBERT physics.
