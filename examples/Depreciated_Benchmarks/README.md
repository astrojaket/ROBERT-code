# Depreciated Benchmarks

This directory preserves superseded benchmark material for historical and
reproducibility purposes. These scripts are not part of the maintained ROBERT
validation workflow and are not run in CI.

ROBERT's maintained gold-standard forward-model validation is based on the
PICASO and petitRADTRANS benchmark scripts in the parent `examples/` directory.

The directory name retains the project's requested `Depreciated_Benchmarks`
spelling.
